Petfinder Machine Learning Project

The idea behind this project was to help shelters determine how "adoptable" their pet was, in hopes to help promote less "adoptable" pets to be adopted. This project utilized a Deep Learning Neural Networks in order to predict scores for pets based on their data from Petfinder. Data was originally brought in from a Kaggle dataset. The web app also utilized PetFinders API in order to bring in new data to help train the model. A user can run data through the model in two different ways. Input fields where the user could enter the data manually. If the pet was already on PetFinder though, the user could enter the pets id and pull the data straight from PetFinders API in order to get the pets adoptability score.

Style Me Website

For this project, the goal was to build a full stack website. The website is based around the idea of building different outfits uploaded from either a picture on the internet or taken from your camera, then saving these outfits to different collections.

The site used several different technologies to deploy. Node.js and express.js were used to create the server. While Sequelize was used for the database structure. The site also utilized two different passport.js stragies for authentication, a local strategy as well as Google oAuth2 authentication.

Mix Tap Maker

The idea for this project was to create a small web app that would create a mix tape for a user based on a band that they input. The User inputs a band and a 10 song mixed tape with related artists is created. Several API's were used for this app. The first API was used to generate the simliar artists list. The second API is used to gather all the information about the artists and the song itself. There is a media player in order for the user to play the songs. The user can also save playlists and come back to them later.
